# Free Subtitle Generator

A 100% free, open-source subtitle generator that creates accurate SRT files from short audio or video files.

## Features
- Automatic language detection
- Supports Hindi, Marathi, Bengali, English & more
- Romanized subtitles for Indian languages
- English preserved as-is
- Accurate timestamps

## Tech Stack
- Python
- Streamlit
- OpenAI Whisper (MIT License)
- Indic Transliteration (AI4Bharat)
- FFmpeg

## Limits
- Best for files under 10 minutes
- CPU-only (free hosting)
